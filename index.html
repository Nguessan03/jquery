I-DEFINITION

jQuery est ce qu’on appelle une « librairie » ou une « bibliothèque » JavaScript. Le rôle d’une librairie, en informatique, est de simplifier l’utilisation d’un certain langage de programmation en fournissant un ensemble de codes déjà prêts à l’emploi. En l’occurrence, la librairie jQuery consiste en un ensemble de blocs de codes JavaScripts préconçus et qui vont être généralement enfermés dans des méthodes. Il va donc nous suffire d’appeler ces méthodes pour exécuter le code qu’elles contiennent.

syntaxe de base de jQuery: $( selector ). geste ()

II-HISTOIRE

La bibliothèque JavaScript libre jQuery est créée en 2006 par le développeur américain John Resig. Alors étudiant à l'université, celui-ci l'a conçue comme un outil facilitant l'exploration d'un document HTML, via sa représentation objet, et permettant de surmonter les différences d'interprétation du code JavaScript par les navigateurs web4.La première version de JQuery est publiée en janvier 2006 ; la version stable 1.0, en août de la même année.
Au départ, l'œuvre d'une seule personne, le projet devient celui d'une communauté de développeurs bénévoles. Il obtient aussi la reconnaissance de la Mozilla Foundation et de multinationales de l'informatique telles que IBM, Google et Microsoft5.

III-CE QUE JQUERY PERMET DE FAIRE

La bibliothèque jQuery permet, entre autres, de gagner en rapidité dans l'interaction avec le code HTML d'une page Web. Elle propose comme principales fonctionnalités :

  • la manipulation du Document Object Model (DOM)
  • la gestion des événements (mouvements de souris, clics, etc.) et de l'AJAX (architecture informatique)
  • la création d'effets d'animation
  • la manipulation des feuilles de style en cascade
  • et bien d'autres encore.

IV-INCLUSION DE JQUERY

1- les formats jQuery

Tout d'abord, vous devrez inclure jQuery dans vos projets web. jQuery est disponible dans deux formats différents : le format de développement, et celui de production. Le fichier développement est le plus gros, car il est annoté de manière à faciliter aux développeurs la compréhension du fonctionnement du code. Il y a des commentaires dans le code, des blancs utiles pour séparer les sections, et d'autres choses encore. Ces annotations, parsemées tout au long du code, vous aident à intégrer jQuery de façon intelligente, comme si vous aviez des conseils directement au sein du code.
Le fichier contenant le format de développement de jQuery est simplement jquery-1.12.2.js(les nombres peuvent différer si vous n'utilisez pas la même version de jQuery).

Le format de production est plus petit, et donc plus rapide à utiliser sur vos pages web en ligne. La plupart des annotations que l'on trouve dans le format de développement ont été retirées, puisque l'"utilisateur" final sera une page web, et non un être humain qui trouverait un intérêt à ces remarques additionnelles dans le code. Le fichier contenant le format de production de jQuery est jquery-1.12.2.min.js . La seule différence dans le nom du fichier est .min, qui indique que le contenu est compressé.

2-inclusion

Pour inclure les fichiers jQuery dans votre code, vous disposez également de deux options (que le site soit en développement ou en production). La première est un scénario classique : il faut télécharger une copie du format de développement ou de production, et l'inclure dans les fichiers de votre projets Mais je vous recommande plutôt la deuxième option, qui consiste à inclure jQuery via CDN (Content Delivery Network). Les fichiers jQuery sont hébergés sur des serveurs du monde entier, vous n'avez donc pas besoin de les mettre sur le vôtre ! Cela permet une meilleure performance, et qu'un utilisateur ait déjà le fichier dans sa mémoire locale ("en cache") après avoir visité un autre site qui utilise la même version de jQuery

V-MANIPULATION DE JQUERY

1-Télécharger jQuery

Il est important de télécharger cette bibliothèque JavaScript pour pouvoir l'utiliser, sinon ce serait impossible. Pour cette raison, la première étape consiste à le télécharger. C'est une étape très simple, car il suffit de passer à le site officiel de jQuery de le faire. Il est recommandé de télécharger la version la plus récente disponible sur la page. Cela vous permettra de profiter des progrès qui ont été réalisés au cours de la dernière année. Par conséquent, vous aurez sûrement plus d'outils et de fonctions à utiliser dans la bibliothèque.

2-Ajoutez jQuery à votre site Web

Maintenant pour utiliser jQuery dans votre page Web, vous devez l'intégrer. Pour cela, vous devez accéder au HTML de celui-ci et saisir ce qui suit pour la balise "Head": En faisant cela, vous ajouterez toutes les informations dont dispose la bibliothèque à votre HTML. Nous en expliquerons un peu plus ci-dessous.

3-Apporter des modifications à votre site Web

En fonction de ce que vous allez faire, l'ordre que vous rédigerez variera. Cependant, dans la plupart des cas, vous devez commencer de la même manière. Pour exécuter des fonctions dans jQuery vous devez placer:

VI-AVANTAGE ET INCONVENIENT

1-Avantages

Le principal avantage de jQuery est qu’il est beaucoup plus facile que ses concurrents. Vous pouvez ajouter des plugins facilement, ce qui se traduit par un gain substantiel de temps et d’efforts. En fait, l’une des principales raisons pour lesquelles Resig et son équipe ont créé jQuery était de gagner du temps (dans le monde du développement web, le temps compte beaucoup). La licence open source de jQuery permet à la bibliothèque d’avoir toujours un support constant et rapide, publiant constamment des mises à jour. La communauté jQuery est active et extrêmement travailleuse.

2-Inconvenients

L’un des principaux inconvénients de jQuery est le grand nombre de versions publiées dans un court laps de temps. Peu importe si vous exécutez la dernière version de jQuery, vous devrez héberger la bibliothèque vous-même (et la mettre à jour constamment), ou télécharger la bibliothèque à partir de Google (attrayant, mais peut apporter des problèmes d’incompatibilité avec le code). En plus du problème des versions, d’autres inconvénients que nous pouvons mentionner: jQuery est facile à installer et à apprendre, au départ. Mais ce n’est pas si facile si nous le comparons avec CSS Si jQuery est mal implémenté en tant que Framework, l’environnement de développement peut devenir incontrôlable.
